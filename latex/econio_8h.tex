Get one raw character from terminal. This can detect F1-\/\+F10, cursor keys, backspace and other controlling keys\+: see the keyboard constants.\+DIW ASCII code is returned for other keys. Non-\/\+ASCII keys probably won\textquotesingle{}t work. Characters are not echoed to the screen when in raw mode. Note that backspace will be code 8, regardless of terminal settings (whether it sent BS or DEL char). Enter will always be 10, even on Windows. On Windows, this function sometimes returns 0\textquotesingle{}s, so just ignore them. Also function keys are not supported on Windows. \mbox{\Hypertarget{econio_8h_a30687da65226b3c30b9903eb30db535d}\label{econio_8h_a30687da65226b3c30b9903eb30db535d}}
