{\footnotesize\ttfamily int econio\+\_\+getch (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

Get one raw character from terminal. This can detect F1-\/\+F10, cursor keys, backspace and other controlling keys\+: see the keyboard constants.\+DIW ASCII code is returned for other keys. Non-\/\+ASCII keys probably won\textquotesingle{}t work. Characters are not echoed to the screen when in raw mode. Only to be used after calling \mbox{\hyperlink{econio_8h_aeca358bf2619bf1c645b9a8068e877b7}{econio\+\_\+rawmode()}}. Note that backspace will be code 8, regardless of terminal settings (whether it sent BS or DEL char). Enter will always be 10, even on Windows. On Windows, this function sometimes returns 0\textquotesingle{}s, so just ignore them. Also function keys are not supported on Windows. \mbox{\Hypertarget{econio_8c_a30687da65226b3c30b9903eb30db535d}\label{econio_8c_a30687da65226b3c30b9903eb30db535d}}
